
# MEDICO Web App



The Medico Hospital Management System is a web-based application designed to 
integrate machine learning technology with healthcare services to assist in disease 
prediction and hospital administration. The system provides predictive models for 
diabetes and heart disease, analysing patient data to offer risk assessments. This feature 
aims to aid in early detection and encourage preventive healthcare measures. By 
incorporating artificial intelligence into medical decision-making, the system enhances 
diagnostic capabilities and supports doctors in providing timely and informed treatment 
plans.

## Run Locally

in order to work the complete project you've to pull the backend also, which is currently not available for public

Clone the project

```bash
  git clone https://github.com/arjunmk4u/medico.git
```

Go to the project directory

```bash
  cd medico
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
